Checkout via lightbox

O checkout via lightbox é o tipo de integração da Gerencianet que funciona como um facilitador para o desenvolvedor.

Neste tipo de integração, o desenvolvedor não precisará implementar uma tela de pagamento para sua aplicação. A mesma será o lightbox.

Para mais informações sobre esta forma de integração, clique aqui ou acesse a documentação da API Gerencianet.
